# ⚛️ React Projects

A monorepo of React projects built while learning React from scratch. Each folder is a standalone project focusing on different concepts and tools.

---

## 📁 Projects

| Project | Description | Tech Used |
|---|---|---|
| `react-components-project` | Isolated components for practicing React fundamentals | React, useState, props |
| `react-notesapp-project` | Notes app with localStorage persistence | React, useState, useEffect |
| `apple-website` | Apple iPhone 15 Pro landing page clone with GSAP animations | React, Tailwind, GSAP |

---

## 🛠️ Tech Stack Across Projects

- **React.js** — UI library
- **Vite** — dev server and bundler
- **Tailwind CSS** — styling
- **GSAP** — animations
- **localStorage** — client-side persistence

---

## ⚙️ Running Any Project

```bash
# Clone the repo
git clone https://github.com/yarnagulagauravdeep-cyber/React-projects.git

# Navigate into any project
cd React-projects/<project-name>

# Install dependencies
npm install

# Start dev server
npm run dev
```
